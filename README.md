## Build

```
cargo build --release
```

## Usage

### 256 Bits

- #### 32-bit word

```
./target/release/u256 32 <HEX>
```

- #### 64-bit word

```
./target/release/u256 64 <HEX>
```

### 384 Bits

- #### 32-bit word

```
./target/release/u381 32 <HEX>
```

- #### 64-bit word

```
./target/release/u381 64 <HEX>
```
